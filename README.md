## 📊 Sistema de Automatización de Ventas y Flujo de Caja para la empresa Xclusive (barbería) 

Desarrollé una solución integral de automatización orientada a la gestión financiera operativa, enfocada en el registro, control y análisis de ingresos y egresos en tiempo real.

El sistema permite la captura estructurada de ventas diferenciadas por **productos y servicios**, así como el registro de **egresos de cualquier naturaleza**, centralizando toda la información en una única fuente de datos confiable.

A partir de esta base, implementé un **flujo de caja automatizado**, el cual se actualiza dinámicamente en función de los movimientos registrados, asegurando consistencia y trazabilidad en la información financiera.

### 🎯 Funcionalidades principales

- Registro automatizado de ventas por **producto y servicio**
- Registro y categorización de **egresos operativos**
- Consolidación de datos en una base centralizada
- Generación automática de **flujo de caja**
- Actualización dinámica en tiempo real

### 📈 Beneficios

- Visualización clara de la **posición de caja** (diaria, semanal y mensual)
- Mayor control sobre la **liquidez del negocio**
- Reducción de errores manuales en el registro de información
- Mejora en la **toma de decisiones basada en datos**
- Ahorro de tiempo mediante automatización de procesos repetitivos

### ⚙️ Tecnologías utilizadas

- **Google Sheets** como sistema de almacenamiento y procesamiento de datos
- **Google Apps Script** para la automatización, validación y actualización de información

### 🚀 Valor agregado

- Integración automática entre ingresos, egresos y flujo de caja
- Escalabilidad para incorporar nuevos tipos de transacciones
- Acceso en tiempo real a información financiera crítica
- Estandarización de procesos financieros operativos
